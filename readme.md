# Save the Pets

Welcome to the Animal Adoption Site!

## Description

The Animal Adoption Site is a platform dedicated to helping animals find loving homes. It provides a user-friendly interface for browsing and adopting animals in need. From playful puppies to cuddly kittens, you can explore a variety of animals waiting to become part of your family.

## Usage

+ Browse animals: Visit the homepage to see a list of available animals
+ View animal details: Click on an animal to view more information about it
+ Submit adoption request: If you're interested in adopting an animal, click on the "Adopt" button and fill out the adoption form

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/mihir9702/savethepets.git

2. Navigate to the project directory:

   ```bash
   cd savethepets

3. Install dependencies:

   ```bash
   pip install -r requirements.txt

4. Apply migrations:
  
   ```bash
   python manage.py migrate

5. Run the development server:
  
   ```bash
   python manage.py runserver

6. Access the application in your web browser at `http://127.0.0.1:8000/`
